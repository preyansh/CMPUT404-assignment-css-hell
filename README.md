Assignment: CSS Hell
====================

You will skin 3 project gutenberg stories with custom CSS.

You will skin 2 versions of a possible professional homepage for your
self with 2 versions of CSS.

Read requirements.org

Read this comic http://theoatmeal.com/comics/design_hell

git clone https://github.com/abramhindle/CMPUT404-assignment-css-hell.git


Part 1:
=======
Books Used:
Peter Pan by J.M. Barrie
The Adventures of Huckleberry Finn by Mark Twain
The Jungle Book by Rudyard Kipling

Modifications in HTML:
Removed CSS provided by Project Gutenberg 
Added own Personal CSS file named myBookStyleSheet.css

What myBookStyleSheet.css changes in HTML:
Changes background to background resembling Parchment Sheet
Changed Font for all text to Helvetica for better reading
Chaged font size so its easier to read
Changed line gap for easier reading
changed margins around text so it looks nicer
Added border around images
Changed Header size
